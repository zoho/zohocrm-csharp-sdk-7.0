Zoho CRM C# SDK 1.0.0
————————————————————————

This is the readme file for Zoho CRM’s C# SDK version 1.0.0.

This file gives a brief of the enhancements and/or bug fixes in the latest version.

----------------
Enhancements
----------------
    - C# SDK upgraded to support v7 APIs.
    - C# SDK improved to support the following new APIs

      - Export Audit Log API
      - Cadence API
      - Users Re-invite API
      - Record Clone API
      - Duplicate Record Check API
      - Data Enrichment API
      - Mass Delete Tags API
      - Picklist Option API
      - API Fetcher

You can also take a look at our GitHub page here (https://github.com/zoho/zohocrm-csharp-sdk-7.0/blob/master/README.md)
